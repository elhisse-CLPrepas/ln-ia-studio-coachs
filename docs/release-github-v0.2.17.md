# Release GitHub - v0.2.17

## Titre

LN-IA Studio Coachs v0.2.17 - Kit atelier complet et sélection des modèles IA

## Résumé

Cette release complète le passage de la landing page vers un kit atelier exploitable. Elle ajoute une sélection claire des modèles IA adaptés à l'activité LN-IA Studio Coachs, un index de kit complet et des supports de facilitation avancés pour accompagner les coachs, managers, formateurs et animateurs.

## Nouveautés

- Ajout du kit atelier complet.
- Ajout d'une fiche de sélection des modèles IA.
- Ajout de supports de facilitation avancés.
- Mise à jour du README avec les nouveaux livrables.
- Mise à jour du changelog pour préparer la release `v0.2.17`.

## Modèles IA recommandés

- GPT-5.2 : modèle principal de production, structuration, prompts, supports, code et release.
- Claude Sonnet 4 : relecture critique, posture humaine, reformulation pédagogique.
- Gemini 2.5 Pro : analyse multimodale, documents longs, PDF, images et supports composites.
- Mistral Large 3 : alternative francophone/open, discussion souveraineté et démonstration européenne.
- Modèles rapides : GPT-5 mini, Gemini Flash ou équivalent pour variantes et tâches simples.
