## 3. New Paradigm IA

Le New Paradigm IA désigne une nouvelle manière de travailler avec l'intelligence artificielle :

- clarifier une intention ;
- dialoguer avec l'IA ;
- produire une première version ;
- vérifier ;
- corriger ;
- publier ou transmettre ;
- documenter ce qui a été appris.
