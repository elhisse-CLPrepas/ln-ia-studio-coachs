# Guide animateur - LN-IA Studio Coachs

## Rôle de l'animateur

L'animateur LN-IA n'est pas seulement un présentateur. Il est un facilitateur d'orchestration.

Son rôle est de :

- donner une vision ;
- rendre les concepts accessibles ;
- relier l'IA aux besoins du public ;
- faire pratiquer ;
- reformuler les questions ;
- garder un cadre humain, éthique et pédagogique ;
- transformer les échanges en prochaines actions.

## Posture

Adopter une posture :

- claire ;
- calme ;
- structurante ;
- encourageante ;
- exigeante sans être technique ;
- orientée impact humain.

Phrase repère :

> Nous ne sommes pas ici pour courir derrière les outils. Nous sommes ici pour apprendre à orchestrer l'IA avec méthode.

## Rythme recommandé

| Moment | Rythme | Intention |
|---|---|---|
| Ouverture | calme et accueillant | installer la confiance |
| Vision | narratif et inspirant | ouvrir le regard |
| New Paradigm IA | pédagogique | clarifier la méthode |
| Démonstration | concret et dynamique | montrer la valeur |
| Discussion | ouvert et attentif | faire émerger les besoins |
| Clôture | synthétique et mobilisateur | donner envie d'agir |

## Gestion du temps

Règle simple :

- ne pas dépasser 60 minutes pour la vision ;
- garder 30 minutes réelles pour l'atelier pratique ;
- préserver 30 minutes pour la discussion ;
- si le temps manque, réduire les exemples, pas la discussion.

## Transitions utiles

### De la vision vers la pratique

> Nous avons compris le changement. Maintenant, regardons comment cela devient une pratique.

### De la pratique vers la discussion

> Ce test montre une méthode. Maintenant, j'aimerais entendre vos besoins réels.

### De la discussion vers la clôture

> Ce que vous venez de partager dessine déjà les prochains ateliers LN-IA.

## Comment gérer les questions

### Question technique

Réponse type :

> C'est une bonne question. Je vais la relier à la méthode : quel est le besoin derrière l'outil ?

### Question inquiète

Réponse type :

> Votre inquiétude est légitime. C'est justement pour cela que nous parlons d'orchestration, d'éthique et de responsabilité humaine.

### Question trop large

Réponse type :

> Je vous propose de la ramener à une situation concrète : dans quel contexte voulez-vous utiliser l'IA ?

## Ce que l'animateur doit éviter

- Présenter l'IA comme une solution magique.
- Faire une démonstration trop longue.
- Multiplier les outils.
- Oublier le public non technique.
- Répondre trop vite sans reformuler.
- Perdre la discussion finale.

## Ce que l'animateur doit toujours faire

- Rappeler l'intention.
- Donner le contexte.
- Montrer la méthode.
- Vérifier la compréhension.
- Relier les exemples aux métiers des participants.
- Faire émerger les besoins.
- Clôturer avec une action claire.

## Clôture recommandée

> Le coach ou le manager augmenté par IA n'est pas celui qui utilise le plus d'outils. C'est celui qui sait orchestrer l'intention, les méthodes, les productions et l'impact humain.

> LN-IA accompagne cette montée en compétence.
