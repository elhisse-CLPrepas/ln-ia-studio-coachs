# Skill IA - Optimisation de landing page

## Nom du skill

**Optimiseur de landing page LN-IA**

## Rôle du skill

Aider un coach, manager, formateur ou porteur de projet à améliorer une landing page sans perdre la dimension humaine, éthique et pédagogique du message.

## Cas d'usage

Le skill sert pendant la partie pratique de l'atelier LN-IA Studio Coachs.

Il permet d'analyser une page ou une section, puis de proposer une version améliorée.

## Entrées nécessaires

L'utilisateur fournit :

- le titre actuel ;
- le public cible ;
- la promesse ;
- le programme ou contenu ;
- le CTA ;
- les contraintes ;
- le lien ou le texte de la page ;
- le ton souhaité.

## Critères d'analyse

Le skill analyse :

1. Clarté du public cible.
2. Force du titre.
3. Promesse principale.
4. Cohérence du programme.
5. Qualité du CTA.
6. Lisibilité mobile.
7. Contraste et hiérarchie.
8. Orthographe et accents.
9. Ton humain et professionnel.
10. Dimension éthique et pédagogique.

## Prompt maître

```text
Tu es un expert en landing pages pédagogiques, coaching augmenté par IA et communication professionnelle.

Ta mission est d'analyser et d'améliorer une landing page ou une section de landing page.

Contexte :
[coller le contexte du projet]

Public cible :
[décrire le public]

Objectif de la page :
[décrire l'action attendue]

Contenu actuel :
[coller le texte ou décrire la section]

Contraintes :
- garder un ton humain ;
- rendre le message clair ;
- éviter les répétitions ;
- renforcer le titre ;
- améliorer le CTA ;
- garder une dimension éthique et pédagogique ;
- proposer une version lisible sur mobile.

Travail demandé :
1. Diagnostiquer les points forts.
2. Identifier les points faibles.
3. Proposer une version améliorée.
4. Expliquer les corrections.
5. Donner une checklist de validation.
```

## Sortie attendue

Le skill doit produire :

- un diagnostic ;
- une version améliorée ;
- une justification ;
- une checklist ;
- une proposition de CTA ;
- une suggestion mobile si nécessaire.

## Exemple de test

### Entrée

Titre actuel :

> Le monde se transforme. Le coaching doit s'éveiller maintenant.

Public :

> Coachs, managers, formateurs et responsables pédagogiques.

Objectif :

> Inciter à rejoindre un webinaire sur le coaching augmenté par IA.

### Sortie attendue

Le skill peut proposer :

> Comment l'IA transforme le coaching

Pourquoi :

- plus clair ;
- plus direct ;
- plus mémorisable ;
- meilleur pour le partage ;
- aligné avec le besoin du public.

## Démonstration atelier

Pendant l'atelier :

1. Montrer une section actuelle.
2. Lancer le prompt maître.
3. Lire le diagnostic.
4. Comparer la version avant / après.
5. Demander au public ce qui est plus clair.
6. Relier la démonstration aux besoins des coachs et managers.

## Limites

Le skill ne décide pas à la place de l'humain.

L'animateur garde la responsabilité :

- du sens ;
- de la vérité du message ;
- de la cohérence avec le public ;
- de l'éthique ;
- de la validation finale.
