# Fiche méthode - Orchestration LN-IA

## Idée centrale

L'orchestration IA consiste à transformer une intention humaine en livrable utile grâce à une suite d'étapes claires : cadrer, dialoguer avec l'IA, produire, vérifier, corriger, publier et apprendre.

## Formule LN-IA

**Intention + Contexte + IA + Vérification + Publication = Impact humain**

## Les 7 gestes de l'orchestrateur IA

### 1. Clarifier l'intention

Question :

> Qu'est-ce que je veux produire, pour qui, et pourquoi ?

Sortie attendue :

- une phrase d'intention claire ;
- un public cible ;
- un résultat concret.

### 2. Donner le contexte

Question :

> Quelles informations l'IA doit-elle connaître pour bien m'aider ?

Sortie attendue :

- contexte ;
- contraintes ;
- ton ;
- format ;
- critères de réussite.

### 3. Dialoguer avec l'IA

Question :

> Quel prompt permet de démarrer la production ?

Sortie attendue :

- un premier prompt ;
- une première réponse ;
- une relance pour améliorer.

### 4. Vérifier

Question :

> Le résultat est-il clair, exact, utile et responsable ?

Sortie attendue :

- points forts ;
- points faibles ;
- risques ;
- corrections nécessaires.

### 5. Corriger

Question :

> Que faut-il modifier pour augmenter la qualité ?

Sortie attendue :

- version améliorée ;
- justification des changements ;
- meilleure lisibilité.

### 6. Publier ou transmettre

Question :

> Où et comment ce livrable doit-il être utilisé ?

Sortie attendue :

- page ;
- fiche ;
- message ;
- slide ;
- checklist ;
- support participant.

### 7. Capitaliser

Question :

> Qu'ai-je appris et que puis-je réutiliser ?

Sortie attendue :

- trace ;
- prompt amélioré ;
- règle de méthode ;
- prochaine action.

## Mini-checklist

- [ ] L'intention est claire.
- [ ] Le public est identifié.
- [ ] Le contexte est donné.
- [ ] Le prompt est précis.
- [ ] Le résultat est vérifié.
- [ ] La correction améliore vraiment la qualité.
- [ ] Le livrable peut être partagé.
- [ ] L'apprentissage est documenté.

## Phrase à retenir

> Orchestrer l'IA, ce n'est pas demander plus vite. C'est produire mieux, avec méthode et responsabilité.
