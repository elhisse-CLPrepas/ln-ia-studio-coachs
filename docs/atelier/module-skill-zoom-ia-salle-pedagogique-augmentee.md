# Module - Skill Zoom IA

## Titre

**Zoom IA : construire une salle pédagogique augmentée**

## Position dans la ligne de production LN-IA

Ce module fait partie de la ligne de production pédagogique LN-IA.

Il complète les modules :

- Orchestration IA ;
- Skills IA ;
- Optimisation de landing page ;
- Process Zoom IA - LN-IA.

## Intention

Former les coachs, managers, formateurs et responsables pédagogiques à maîtriser Zoom non plus comme un simple outil de visioconférence, mais comme une salle pédagogique augmentée.

## Idée centrale

Zoom devient un skill stratégique du coach augmenté IA.

Maîtriser Zoom, c'est apprendre à :

- préparer une session ;
- animer ;
- capter les interactions ;
- utiliser les fonctions IA ;
- transformer les traces ;
- produire des supports ;
- prolonger l'apprentissage ;
- améliorer l'atelier suivant.

## Promesse du module

À la fin du module, le participant sait organiser une session Zoom comme un environnement d'apprentissage augmenté, capable de produire des traces, des synthèses, des besoins, des actions et des livrables.

## Public cible

- Coachs.
- Managers.
- Formateurs.
- Enseignants.
- Responsables pédagogiques.
- Animateurs de communauté.

## Objectifs pédagogiques

À la fin du module, le participant sera capable de :

- préparer une session Zoom avec une logique pédagogique ;
- identifier les fonctions utiles de Zoom et Zoom IA ;
- organiser le chat et les interactions ;
- capter les questions et besoins du public ;
- exploiter les enregistrements, résumés ou transcriptions ;
- transformer une session en supports ;
- relier Zoom à une logique GitHub / documentation ;
- améliorer une formation à partir des traces.

## Compétences travaillées

### 1. Préparer

- Créer le lien Zoom.
- Vérifier les paramètres.
- Préparer la landing page.
- Préparer les supports.
- Préparer les prompts.
- Préparer la fiche de collecte.

### 2. Animer

- Accueillir.
- Partager l'écran.
- Gérer le rythme.
- Utiliser le chat.
- Faire participer.
- Reformuler.
- Garder la trace des besoins.

### 3. Capter

- Enregistrer.
- Sauvegarder le chat.
- Récupérer les questions.
- Utiliser les résumés IA si disponibles.
- Exploiter la transcription si disponible.

### 4. Transformer

- Produire un compte-rendu.
- Extraire une FAQ.
- Identifier les besoins.
- Créer de nouveaux supports.
- Préparer une prochaine offre.

### 5. Versionner

- Mettre à jour les fichiers du projet.
- Committer.
- Taguer.
- Pousser vers GitHub.
- Garder une mémoire de progression.

## Déroulé recommandé

Durée : 60 à 90 minutes.

| Séquence | Durée | Contenu |
|---|---:|---|
| Vision | 10 min | Zoom comme salle pédagogique augmentée |
| Préparation | 15 min | Réglages, supports, liens, prompts |
| Animation | 15 min | Rythme, chat, questions, engagement |
| Captation | 15 min | Enregistrement, résumé, transcription, chat |
| Transformation | 20 min | Compte-rendu, FAQ, supports, GitHub |
| Synthèse | 5 min | Prochaine action du participant |

## Activité pratique

### Exercice : préparer une salle augmentée

Le participant choisit une session à animer et complète :

- titre ;
- public ;
- objectif ;
- lien Zoom ;
- supports à ouvrir ;
- fonctions Zoom à activer ;
- questions à poser ;
- traces à récupérer ;
- livrable post-session.

## Checklist express

- [ ] Le lien Zoom est prêt.
- [ ] La landing page est ouverte.
- [ ] Les supports sont ouverts.
- [ ] Le chat est prévu comme source de questions.
- [ ] L'enregistrement est décidé.
- [ ] Les fonctions IA disponibles sont identifiées.
- [ ] La fiche de collecte des besoins est prête.
- [ ] Le compte-rendu post-atelier est prévu.
- [ ] Le dépôt GitHub recevra les livrables.

## Message clé

> Le coach augmenté IA ne se contente pas d'animer une réunion Zoom. Il transforme Zoom en salle pédagogique augmentée, puis transforme les traces en apprentissages, supports et actions.

## Valeur ajoutée LN-IA

LN-IA aide les coachs et managers à faire de Zoom :

- un espace d'apprentissage ;
- un outil de captation ;
- une source de données pédagogiques ;
- un moteur de production de supports ;
- un point d'entrée vers l'orchestration IA.
