# Checklist animateur - LN-IA Studio Coachs

## Avant l'atelier

- Vérifier que la landing page est accessible.
- Vérifier le lien WhatsApp.
- Vérifier le lien Zoom.
- Préparer les affiches et visuels.
- Ouvrir les documents de cadrage.
- Préparer un exemple de prompt simple.
- Préparer un exemple de section de landing page à améliorer.
- Vérifier l'affichage mobile de la landing page.

## Ouverture

- Accueillir les participants.
- Présenter le thème : comment l'IA transforme le coaching.
- Rappeler la promesse : passer du coaching classique au coaching augmenté par IA sans perdre l'humain.
- Expliquer que l'atelier est pratique, progressif et versionnable.

## Pendant l'atelier

- Garder un rythme clair.
- Relier chaque idée à une pratique de coaching.
- Ne pas présenter l'IA comme une solution magique.
- Montrer comment vérifier et améliorer les productions.
- Faire verbaliser les décisions humaines.
- Insister sur l'éthique, la pédagogie et la responsabilité.

## Points de vigilance

- Éviter les démonstrations trop techniques.
- Éviter de surcharger les participants avec trop d'outils.
- Privilégier une progression simple : intention, prompt, résultat, correction.
- Garder la landing page comme fil conducteur.
- Rappeler que le coach reste responsable du cadre.

## Après l'atelier

- Partager la landing page.
- Partager le message WhatsApp.
- Partager le message Facebook ou LinkedIn.
- Noter les questions récurrentes.
- Mettre à jour la FAQ si nécessaire.
- Préparer le prochain commit Git si des supports évoluent.

## Validation qualité

L'atelier est prêt si :

- le participant comprend le New Paradigm IA ;
- les activités sont réalisables sans outil complexe ;
- les supports sont lisibles ;
- le CTA WhatsApp fonctionne ;
- la page publiée reste cohérente avec le message oral ;
- les prochaines améliorations sont documentées.
