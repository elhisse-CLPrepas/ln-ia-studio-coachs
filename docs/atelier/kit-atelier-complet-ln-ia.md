# Kit atelier complet - LN-IA Studio Coachs

Ce kit rassemble les supports nécessaires pour préparer, animer, faciliter, documenter et publier un atelier LN-IA Studio Coachs.

## Objectif

Permettre à un animateur de conduire une session claire autour du coaching augmenté par IA :

- comprendre le changement IA ;
- adopter une posture d'orchestration ;
- produire un premier livrable concret ;
- faciliter les échanges ;
- transformer la séance en suite opérationnelle.

## Parcours animateur

1. Lire le programme : `programme-formation-ln-ia-sc.md`.
2. Préparer la session : `deroule-atelier-ln-ia-sc.md` et `session-2h-ln-ia-sc.md`.
3. Choisir les modèles IA : `selection-modeles-ia-ln-ia.md`.
4. Préparer les participants : `fiche-participant-ln-ia-sc.md`.
5. Animer : `script-animation-2h-ln-ia-sc.md`.
6. Faciliter : `guide-animateur-ln-ia-sc.md`, `cartes-facilitation-ln-ia.md` et `supports-facilitation-avances-ln-ia.md`.
7. Faire pratiquer : `canevas-exercice-orchestration-ia.md` et `grille-analyse-landing-page.md`.
8. Évaluer : `evaluation-formation-ln-ia-sc.md`.
9. Capitaliser : `compte-rendu-post-atelier.md`.
10. Publier : `release-github-v0.2.17.md`.

## Livrables de sortie

À la fin de l'atelier, l'animateur doit pouvoir produire :

- une synthèse des besoins du groupe ;
- une grille d'analyse d'une landing page ;
- un premier prompt amélioré ;
- un plan d'action personnel ;
- un compte-rendu post-atelier ;
- une note de release GitHub si des supports ont été publiés.
