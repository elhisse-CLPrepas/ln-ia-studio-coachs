# Module - Orchestration IA

## Position dans la formation

Ce module peut être intégré dans la première partie de la session 2h, après l'introduction du New Paradigm IA.

Durée recommandée : 15 à 20 minutes.

## Titre du module

**De l'usage de l'IA à l'orchestration IA**

## Objectif

Faire comprendre que la valeur ajoutée du coach ou du manager ne réside pas seulement dans l'utilisation d'un outil IA, mais dans sa capacité à orchestrer un processus complet.

## Message clé

L'IA accélère la production. L'orchestration donne du sens, de l'ordre, de la qualité et de l'impact humain.

## Définitions

### Utiliser l'IA

Utiliser l'IA, c'est demander une réponse ou une production.

Exemples :

- demander un texte ;
- résumer un document ;
- générer une liste d'idées ;
- corriger une phrase.

### Orchestrer avec l'IA

Orchestrer avec l'IA, c'est organiser une chaîne complète :

1. Intention.
2. Contexte.
3. Sources.
4. Prompt.
5. Production.
6. Vérification.
7. Correction.
8. Publication.
9. Retour d'expérience.

## Exemple simple

### Usage simple

> Écris-moi un message WhatsApp.

### Orchestration

> Voici mon public, mon objectif, mon contexte, ma landing page, mon ton, mes contraintes et le résultat attendu. Propose trois versions, explique les différences, puis aide-moi à choisir la plus adaptée.

## Rôle du coach orchestrateur

Le coach augmenté IA :

- clarifie le besoin ;
- garde la responsabilité du cadre ;
- choisit les bons prompts ;
- évalue les réponses ;
- relie l'outil au contexte humain ;
- transforme la production en apprentissage.

## Rôle du manager orchestrateur

Le manager augmenté IA :

- structure les priorités ;
- organise les livrables ;
- vérifie la qualité ;
- facilite la collaboration ;
- accélère sans perdre la clarté ;
- transforme les idées en actions suivies.

## Activité pédagogique

### Exercice : passer d'un usage simple à une orchestration

Demander aux participants de choisir une demande simple :

- écrire un message ;
- préparer une réunion ;
- créer une fiche ;
- améliorer une page ;
- synthétiser un document.

## Prompt d'exercice

```text
Je veux transformer une demande simple en orchestration IA.

Demande initiale :
[écrire la demande]

Aide-moi à préciser :
1. l'intention ;
2. le public ;
3. le contexte ;
4. les sources utiles ;
5. les contraintes ;
6. le format attendu ;
7. les critères de qualité ;
8. la manière de vérifier le résultat ;
9. l'action à faire après la production.
```

## Débrief

Questions à poser :

1. Qu'est-ce qui a changé entre la demande simple et l'orchestration ?
2. Quelle information a le plus amélioré la qualité de la réponse ?
3. Quel rôle humain reste indispensable ?
4. Comment ce processus peut-il aider un coach ou un manager ?

## Conclusion du module

Un professionnel augmenté par IA n'est pas celui qui demande le plus vite. C'est celui qui orchestre le mieux.

**LN-IA forme à cette orchestration : intention, méthode, production, vérification, transmission et impact humain.**
