# Checklist technique webinaire

## 24 heures avant

- Vérifier le lien Zoom.
- Vérifier le groupe WhatsApp.
- Tester la landing page publiée.
- Vérifier les images de la landing page.
- Préparer les documents atelier.
- Ouvrir le skill d'optimisation.
- Préparer l'exemple à tester.
- Vérifier le navigateur.
- Préparer une copie locale des liens.

## 30 minutes avant

- Ouvrir Zoom.
- Tester micro.
- Tester caméra.
- Tester partage écran.
- Ouvrir la landing page.
- Ouvrir le conducteur 2h.
- Ouvrir le script d'animation.
- Ouvrir la fiche de collecte des besoins.
- Ouvrir le skill IA.
- Fermer les fenêtres inutiles.

## Pendant le webinaire

- Lancer l'enregistrement si nécessaire.
- Garder le chat visible.
- Noter les questions.
- Surveiller le temps.
- Vérifier que le partage écran est lisible.
- Coller le lien WhatsApp au bon moment.
- Garder une version de secours du prompt.

## Après le webinaire

- Sauvegarder le chat.
- Sauvegarder les questions.
- Compléter la fiche de collecte des besoins.
- Noter les problèmes techniques.
- Mettre à jour la FAQ si nécessaire.
- Préparer les prochaines actions.
