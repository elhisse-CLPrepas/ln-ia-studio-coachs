# Déroulé atelier - LN-IA Studio Coachs

## Format recommandé

Durée cible : 2h00 à 2h30.

Modalité : en ligne, avec démonstration et participation active.

Support d'entrée : landing page publiée du webinaire LN-IA Studio Coachs.

## Séquence 1 - Ouverture

Durée : 10 minutes.

Objectifs :

- accueillir les participants ;
- rappeler l'intention de l'atelier ;
- présenter la question centrale : comment l'IA transforme le coaching ?

Message clé :

L'enjeu n'est pas de remplacer le coach. L'enjeu est d'apprendre à travailler avec l'IA sans perdre l'humain.

## Séquence 2 - Changement à l'échelle du monde

Durée : 25 minutes.

Objectifs :

- situer l'IA dans une transformation plus large ;
- relier cloud, satellites, horloges atomiques et quantique à la transformation des métiers ;
- ouvrir une vision systémique.

Activité :

Chaque participant complète la phrase :

> Dans mon métier, le changement le plus visible aujourd'hui est...

## Séquence 3 - Changement personnel et New Paradigm IA

Durée : 30 minutes.

Objectifs :

- montrer que l'IA demande un changement de mindset ;
- introduire le New Paradigm IA ;
- distinguer usage passif et usage structuré.

Activité :

Transformer une intention floue en intention claire :

- intention initiale ;
- clarification ;
- prompt ;
- résultat attendu ;
- critère de vérification.

## Séquence 4 - Skills IA du développement moderne

Durée : 30 minutes.

Objectifs :

- identifier les compétences indispensables ;
- montrer le cycle idée, production, test, correction, publication ;
- relier ces compétences aux pratiques de coaching.

Skills IA à travailler :

- formuler une intention ;
- dialoguer avec un modèle ;
- structurer une production ;
- vérifier la qualité ;
- corriger ;
- publier ;
- documenter.

## Séquence 5 - Échantillon pratique : landing page

Durée : 35 minutes.

Objectifs :

- analyser une landing page ;
- améliorer un titre, un CTA ou une section ;
- comprendre comment l'IA aide à produire sans décider à la place de l'humain.

Activité :

Les participants choisissent une section :

- titre ;
- programme ;
- intervenant ;
- FAQ ;
- message WhatsApp ;
- message Facebook.

Ils proposent une amélioration et expliquent pourquoi.

## Séquence 6 - Synthèse et passage à l'action

Durée : 15 minutes.

Objectifs :

- faire émerger les apprentissages ;
- identifier une action concrète à réaliser après l'atelier ;
- inviter à rejoindre la communauté.

Question de sortie :

> Quelle compétence IA allez-vous pratiquer cette semaine ?

## Livrables de fin d'atelier

- Une intention claire.
- Un prompt amélioré.
- Une section ou un message optimisé.
- Une action personnelle de progression.
