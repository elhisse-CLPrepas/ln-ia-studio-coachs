# Prompts d'atelier - LN-IA Studio Coachs

Ces prompts servent de base de travail pendant la formation. Ils doivent être adaptés au contexte de chaque participant.

## Prompt 1 - Clarifier une intention

```text
Je veux transformer une idée floue en intention claire.

Contexte :
[décrire le métier, le public, la situation]

Idée initiale :
[écrire l'idée brute]

Aide-moi à formuler :
1. une intention claire ;
2. un objectif concret ;
3. un livrable attendu ;
4. trois critères de qualité ;
5. une première action réalisable aujourd'hui.
```

## Prompt 2 - Comprendre un changement professionnel

```text
Agis comme un conseiller pédagogique spécialisé dans l'IA.

Analyse le changement suivant :
[décrire le changement observé]

Explique :
1. ce qui change à l'échelle du monde ;
2. ce qui change à l'échelle personnelle ;
3. ce qui change à l'échelle professionnelle ;
4. les risques à éviter ;
5. les skills IA à développer.

Réponds dans un style clair, structuré et accessible à des coachs.
```

## Prompt 3 - Améliorer une posture de coach augmenté

```text
Je suis coach ou accompagnateur.

Voici une situation :
[décrire la situation d'accompagnement]

Aide-moi à préparer une intervention augmentée par IA sans perdre la dimension humaine.

Donne-moi :
1. les questions à poser ;
2. les apports possibles de l'IA ;
3. les limites à respecter ;
4. les points éthiques ;
5. une trace écrite à produire après la séance.
```

## Prompt 4 - Analyser une landing page

```text
Analyse cette landing page :
[coller le contenu ou décrire la page]

Évalue :
1. le public cible ;
2. la promesse ;
3. le titre ;
4. le programme ;
5. les appels à l'action ;
6. la cohérence des visuels ;
7. la lisibilité mobile ;
8. la qualité humaine et éthique du message.

Propose ensuite 5 améliorations prioritaires.
```

## Prompt 5 - Produire un message de partage

```text
Je veux partager un webinaire sur WhatsApp, Facebook ou LinkedIn.

Informations :
- Titre :
- Public :
- Promesse :
- Date :
- Lien du site :
- Lien WhatsApp :

Produis :
1. un message court pour WhatsApp ;
2. un message plus professionnel pour Facebook ou LinkedIn ;
3. une phrase de conclusion forte ;
4. une version plus chaleureuse ;
5. une version plus institutionnelle.
```

## Prompt 6 - Vérifier une production IA

```text
Voici une production générée avec l'aide de l'IA :
[coller la production]

Vérifie :
1. la clarté ;
2. la cohérence ;
3. les fautes ;
4. les répétitions ;
5. le ton ;
6. la valeur pour le public cible ;
7. les risques de formulation ;
8. les améliorations possibles.

Donne une version corrigée et explique les principales corrections.
```
