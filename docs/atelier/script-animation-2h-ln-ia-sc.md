# Script d'animation 2h - LN-IA Studio Coachs

## Objectif du script

Donner à l'animateur un fil de parole simple, clair et rythmé pour conduire la session de 2 heures.

## Avant ouverture

À préparer à l'écran :

- landing page LN-IA ;
- dossier source de la partie 1 ;
- skill d'optimisation de landing page ;
- exemple à tester ;
- fiche de collecte des besoins ;
- lien WhatsApp ;
- lien Zoom.

## 0:00 - 0:05 | Accueil

Message possible :

> Bonjour à toutes et à tous. Bienvenue dans cette session LN-IA Studio Coachs : Comment l'IA transforme le coaching, le management et le développement moderne.

Objectif :

- accueillir ;
- installer le ton ;
- rappeler que la session est pratique et orientée action.

## 0:05 - 0:10 | Promesse de la session

Message possible :

> En deux heures, nous allons faire trois choses : comprendre le changement, voir une démonstration concrète avec un skill IA, puis discuter des besoins réels des coachs et managers.

À dire clairement :

- l'IA n'est pas seulement un outil ;
- le coach et le manager deviennent des orchestrateurs ;
- LN-IA propose une méthode pour produire et apprendre.

## 0:10 - 0:25 | Changement à l'échelle du monde

Points à couvrir :

- IA générative ;
- cloud ;
- satellites ;
- horloges atomiques ;
- physique quantique ;
- accélération des usages.

Phrase clé :

> Le monde devient plus connecté, plus rapide et plus calculé. Le rôle du professionnel est d'apprendre à lire ce changement.

Interaction courte :

> Quel changement technologique voyez-vous déjà dans votre métier ?

## 0:25 - 0:40 | Changement personnel

Points à couvrir :

- mindset ;
- nouvelles habitudes ;
- intention ;
- prompt ;
- vérification ;
- correction.

Phrase clé :

> Le premier changement n'est pas technique. Il est mental : apprendre à travailler avec l'IA demande une nouvelle discipline de pensée.

## 0:40 - 0:50 | New Paradigm IA

Présenter le cycle :

1. Intention.
2. Contexte.
3. Prompt.
4. Production.
5. Vérification.
6. Correction.
7. Publication.
8. Apprentissage.

Phrase clé :

> L'IA ne remplace pas la pensée. Elle oblige à mieux penser.

## 0:50 - 1:00 | Orchestration IA

Message possible :

> Utiliser l'IA, c'est demander une réponse. Orchestrer l'IA, c'est organiser une chaîne complète : intention, contexte, production, vérification, correction, publication et apprentissage.

Formule :

> Un professionnel augmenté par IA n'est pas celui qui demande le plus vite. C'est celui qui orchestre le mieux.

Transition :

> Nous avons posé la vision. Maintenant, nous allons voir une pratique concrète.

## 1:00 - 1:05 | Présenter le cas pratique

Cas :

Optimiser une landing page.

Dire :

> Une landing page est un bon exemple parce qu'elle oblige à clarifier le public, la promesse, le programme, le CTA, les visuels et la lisibilité mobile.

## 1:05 - 1:10 | Présenter le skill IA

Expliquer :

- rôle du skill ;
- critères d'analyse ;
- sortie attendue ;
- responsabilité humaine.

Phrase clé :

> Un skill IA utile est une méthode encapsulée.

## 1:10 - 1:20 | Test en ligne

Déroulé :

1. Coller ou décrire une section.
2. Lancer le prompt maître.
3. Lire le diagnostic.
4. Repérer les améliorations.
5. Demander au public ce qui est plus clair.

## 1:20 - 1:25 | Comparer avant / après

Questions :

- Le titre est-il plus clair ?
- Le public est-il mieux ciblé ?
- Le CTA est-il plus fort ?
- Le texte est-il plus humain ?
- Le résultat est-il utilisable ?

## 1:25 - 1:30 | Extraire la méthode

Message possible :

> Ce que nous venons de faire n'est pas seulement une optimisation de page. C'est une orchestration : intention, contexte, IA, vérification, correction et livrable.

## 1:30 - 1:55 | Discussion guidée

Utiliser les questions suivantes :

1. Quel changement lié à l'IA vous touche déjà dans votre métier ?
2. Quelle compétence IA devient prioritaire pour vous aujourd'hui ?
3. Qu'est-ce qui vous inquiète le plus dans l'usage de l'IA ?
4. Où l'IA peut-elle réellement aider un coach ou un manager ?
5. De quelle solution concrète auriez-vous besoin après ce webinaire ?

Méthode :

- écouter ;
- reformuler ;
- noter les besoins ;
- relier chaque besoin à une solution LN-IA.

## 1:55 - 2:00 | Clôture

Message de clôture :

> LN-IA - Laboratoire Numérique Intelligence Artificielle propose une méthode pour apprendre, produire, tester et transmettre avec l'IA. Notre ambition est simple : former les coachs et managers aux Skills IA et à l'orchestration du développement moderne.

Slogan :

> LN-IA : apprendre à orchestrer l'IA pour amplifier l'impact humain.

Dernière action :

- inviter à rejoindre le groupe WhatsApp ;
- annoncer la suite ;
- remercier les participants.
