# Dossier Zoom Workplace - LN-IA

## Objet du dossier

Ce dossier analyse le profil Zoom Workplace utilise pour les seances LN-IA et propose une organisation pratique pour transformer Zoom en salle pedagogique augmentee.

Le scan montre un compte Zoom avec licence active, adapte a l'animation de webinaires, d'ateliers et de formations en ligne pour coachs, managers, formateurs et responsables pedagogiques.

## Lecture strategique

Votre abonnement Zoom ne sert pas seulement a lancer une reunion. Il donne acces a un environnement de travail complet.

Pour LN-IA, cela signifie que Zoom peut devenir :

- une salle de webinaire ;
- une salle d'atelier ;
- un espace de captation des besoins ;
- une source de traces pedagogiques ;
- un outil de production de supports apres seance ;
- un point d'entree vers une ligne de production IA.

## Points forts pour LN-IA

### Capacite suffisante pour un atelier

La capacite de 100 participants est largement suffisante pour les premieres sessions LN-IA, les ateliers pilotes, les formations de coachs et les groupes de managers.

### Duree confortable

La limite de 30 heures par reunion permet d'organiser sans contrainte :

- des webinaires de 1h a 2h ;
- des ateliers de 2h ;
- des formations longues ;
- des sessions de suivi.

### Ecosysteme Workplace

Zoom Chat, Scheduler, Mail/Calendar, Clips Plus et AI Docs permettent de structurer l'avant, le pendant et l'apres seance.

Cela s'aligne bien avec la methode LN-IA :

1. preparer ;
2. animer ;
3. capter ;
4. transformer ;
5. documenter ;
6. ameliorer.

## Vigilance securite

Informations a proteger :

- numero de telephone ;
- email de connexion ;
- numero de compte ;
- lien personnel de reunion ;
- cle d'hote ;
- identifiant personnel de reunion ;
- lien de chat direct.

## Recommandations immediates

### Ne pas utiliser le lien personnel pour les webinaires publics

Le lien personnel de reunion doit rester reserve a des usages prives ou controles.

Pour les webinaires LN-IA, il est preferable de creer une reunion dediee par evenement.

### Activer une securite minimale

Pour chaque reunion publique :

- activer une salle d'attente ;
- utiliser un code secret ;
- limiter le partage d'ecran a l'hote au debut ;
- verifier les droits d'enregistrement ;
- ne jamais communiquer la cle d'hote ;
- eviter de publier le lien personnel permanent.

### Renforcer le compte

Actions recommandees :

- activer l'authentification en deux etapes ;
- ajouter une cle d'acces si possible ;
- verifier les appareils connectes ;
- fermer les sessions inconnues ;
- separer le lien public du lien personnel.

## Process recommande pour une seance LN-IA

### Avant la seance

- Creer une reunion dediee dans Zoom Scheduler.
- Verifier le fuseau horaire Casablanca.
- Activer le code secret.
- Activer la salle d'attente si le public est ouvert.
- Preparer la landing page.
- Preparer le support animateur.
- Preparer le plan slides.
- Preparer la fiche participant.
- Preparer le document de collecte des questions.

### Pendant la seance

- Accueillir les participants.
- Rappeler le cadre.
- Indiquer l'usage du chat.
- Partager l'ecran.
- Animer le programme.
- Capter les questions.
- Noter les besoins.
- Garder une cloture claire avec appel a l'action.

### Apres la seance

- Sauvegarder le chat.
- Recuperer le replay si enregistrement active.
- Recuperer les syntheses ou documents IA si disponibles.
- Produire un compte-rendu.
- Extraire une FAQ.
- Identifier les besoins recurrents.
- Mettre a jour les supports LN-IA.
- Preparer le prochain atelier.

## Usage des fonctions IA Zoom

### AI Docs

AI Docs peut servir a structurer :

- compte-rendu de seance ;
- synthese des questions ;
- plan d'action ;
- document de suivi ;
- base de ressources.

### Clips Plus

Clips Plus peut servir a produire :

- extraits courts ;
- capsules pedagogiques ;
- messages de recapitulatif ;
- demonstrations partageables.

### Reconnaissance intelligente

La reconnaissance vocale et les etiquettes intelligentes peuvent aider a mieux identifier les intervenants dans les transcriptions et les resumes.

Avant activation, il faut verifier :

- le niveau de confidentialite souhaite ;
- le consentement des participants ;
- les parametres disponibles dans le compte.

## Positionnement LN-IA

Zoom Workplace devient un module de competence dans la ligne LN-IA.

Un coach augmente IA doit apprendre a maitriser :

- la salle virtuelle ;
- le rythme pedagogique ;
- les interactions ;
- la captation des traces ;
- la transformation en livrables ;
- la publication ou le partage controle.

## Decision recommandee

Pour les prochains webinaires LN-IA :

1. ne pas utiliser le lien personnel permanent ;
2. creer un lien Zoom dedie par evenement ;
3. preparer un kit support complet ;
4. capter les questions et besoins ;
5. produire un compte-rendu post-seance ;
6. versionner les supports utiles dans le projet LN-IA.

## Phrase de synthese

Zoom Workplace n'est pas seulement une plateforme de reunion. Pour LN-IA, c'est une salle pedagogique augmentee capable de relier animation, IA, captation, documentation et production de valeur.
