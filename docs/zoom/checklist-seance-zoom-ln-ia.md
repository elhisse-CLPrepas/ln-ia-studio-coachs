# Checklist seance Zoom - LN-IA

## Avant la seance

- [ ] Lien Zoom cree et teste.
- [ ] Date et heure confirmees.
- [ ] Lien WhatsApp prepare.
- [ ] Landing page ouverte.
- [ ] Slides ou support animateur ouverts.
- [ ] Exemple de demonstration pret.
- [ ] Fiche participant disponible.
- [ ] Fichier de collecte des questions pret.
- [ ] Micro teste.
- [ ] Camera testee.
- [ ] Partage d'ecran teste.
- [ ] Enregistrement decide.
- [ ] Fonctionnalites IA Zoom identifiees.

## Au demarrage

- [ ] Accueillir les participants.
- [ ] Rappeler le titre.
- [ ] Presenter le programme.
- [ ] Preciser l'usage du chat.
- [ ] Indiquer si la seance est enregistree.
- [ ] Donner le lien WhatsApp.

## Pendant la seance

- [ ] Surveiller le chat.
- [ ] Noter les questions importantes.
- [ ] Garder le rythme.
- [ ] Faire des transitions courtes.
- [ ] Reformuler les besoins.
- [ ] Montrer la demonstration en direct.
- [ ] Garder 20 a 30 minutes pour discussion.

## Avant la cloture

- [ ] Resumer les idees fortes.
- [ ] Donner une prochaine action.
- [ ] Rappeler le lien WhatsApp.
- [ ] Inviter au prochain atelier.
- [ ] Remercier les participants.

## Apres la seance

- [ ] Sauvegarder le chat.
- [ ] Recuperer l'enregistrement si disponible.
- [ ] Recuperer le resume IA si disponible.
- [ ] Produire un compte-rendu.
- [ ] Extraire une FAQ.
- [ ] Identifier les besoins du public.
- [ ] Mettre a jour les supports LN-IA.
- [ ] Preparer le message post-seance.
