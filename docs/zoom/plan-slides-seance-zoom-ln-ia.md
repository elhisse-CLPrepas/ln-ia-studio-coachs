# Plan slides - Seance Zoom LN-IA

## Intention

Ce plan sert de base a un support visuel simple pour une seance Zoom de deux heures.

Le support doit rester clair, lisible et vivant. Les slides ne remplacent pas la parole de l'animateur : elles donnent le fil, les mots cles et les images fortes.

## Slide 1 - Titre

**Comment l'IA transforme le coaching, le management et le developpement moderne**

Sous-titre :

Atelier LN-IA Studio Coachs - vers le coach et le manager augmentes IA.

## Slide 2 - Promesse

En deux heures :

- comprendre le changement IA ;
- voir une demonstration pratique ;
- identifier les skills IA indispensables ;
- repartir avec une prochaine action.

## Slide 3 - Programme

1. Vision : le changement IA.
2. Atelier : optimisation d'une landing page.
3. Discussion : besoins, questions et suites LN-IA.

## Slide 4 - Le changement a l'echelle du monde

Mots cles :

- IA generative ;
- cloud ;
- donnees ;
- automatisation ;
- satellites et synchronisation ;
- informatique quantique en horizon.

Message :

Le monde devient plus connecte, plus rapide et plus calcule. Les metiers d'accompagnement doivent apprendre a lire cette transformation.

## Slide 5 - Le changement personnel

Mots cles :

- mindset ;
- intention ;
- prompt ;
- verification ;
- correction ;
- apprentissage continu.

Message :

Travailler avec l'IA demande une nouvelle discipline mentale.

## Slide 6 - New Paradigm IA

Cycle :

1. Intention.
2. Prompt.
3. Production.
4. Verification.
5. Correction.
6. Publication.
7. Apprentissage.

Message :

L'IA ne remplace pas la pensee. Elle oblige a mieux penser.

## Slide 7 - Le coach et le manager augmentes IA

Ils apprennent a relier :

- besoin humain ;
- methode ;
- outil IA ;
- livrable ;
- verification ;
- impact.

## Slide 8 - Demonstration atelier

Cas pratique :

Optimiser une section de landing page avec un skill IA prepare.

Objectif :

Montrer comment passer d'un brouillon a une version plus claire, plus ciblee et plus publiable.

## Slide 9 - Methode de demonstration

1. Lire le contexte.
2. Identifier le public.
3. Clarifier la promesse.
4. Produire une amelioration.
5. Verifier le message.
6. Corriger.

## Slide 10 - Discussion guidee

Questions :

- Quel usage IA est prioritaire pour vous ?
- Quel livrable voulez-vous produire ?
- Quel frein rencontrez-vous ?
- Quel skill IA voulez-vous developper ?

## Slide 11 - Valeur ajoutee LN-IA

LN-IA aide a :

- apprendre ;
- pratiquer ;
- produire ;
- versionner ;
- publier ;
- ameliorer.

## Slide 12 - Appel a l'action

Prochaine etape :

- rejoindre le groupe WhatsApp ;
- proposer un cas concret ;
- participer au prochain atelier ;
- tester un skill IA.

## Slide 13 - Cloture

Phrase forte :

> A l'ere de l'acceleration, le coach augmente IA devient un orchestrateur de sens, de methode et de production.
