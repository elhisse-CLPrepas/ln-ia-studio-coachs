# Mission Codex - LN-IA-SC v0.1.0

## Role

Tu es Codex dans VS Code. Tu aides a transformer le dossier squelette `ln-ia-studio-coachs` en premiere base propre, versionnable et publiable du projet **LAB-NUMERIQUE-IA - Studio Coachs**.

## Limite stricte

La mission est limitee a `v0.1.0`.

Ne pas produire une application complete. Ne pas ajouter de framework, de dependances, de back-end, de base de donnees, d'authentification ou d'automatisation avancee.

## Objectif v0.1.0

Produire une base statique claire :

- README de cadrage.
- Arborescence rangee.
- Documents sources places dans `docs/source/`.
- Images sources placees dans `assets/img/`.
- Page `index.html` minimale et lisible.
- Donnees JSON valides dans `data/`.
- Message de partage dans `share/whatsapp.txt`.
- Roadmap et changelog synchronises avec la portee `v0.1.0`.

## Sources a respecter

S'appuyer sur les documents suivants :

- `guide_nouveaux_comportemant.docx`
- `document_technique_depot_LN_IA_SC.docx`
- `guide_orchestration_LN_IA_SC.docx`
- `guide_chef_atelier.docx`

Themes centraux :

- coaching augmente par IA ;
- apprentissage de nouveaux comportements ;
- humain augmente par IA ;
- atelier professionnel pour coachs ;
- production versionnee avec VS Code, Codex, Git et GitHub.

## Livrables attendus

1. Verifier que les fichiers sources reels sont copies dans `docs/source/`.
2. Verifier que les visuels reels sont disponibles dans `assets/img/`.
3. Mettre a jour `index.html`, `assets/css/style.css` et `assets/js/app.js` uniquement si necessaire pour une page statique simple.
4. Initialiser les JSON avec quelques entrees pedagogiques courtes et valides.
5. Mettre a jour `ROADMAP.md` et `CHANGELOG.md`.

## Criteres de validation

- Le depot reste simple et comprehensible.
- Tous les JSON sont valides.
- La page peut etre ouverte localement sans serveur.
- Aucun fichier source utilisateur n'est supprime.
- Les changements sont faciles a relire avant commit.
